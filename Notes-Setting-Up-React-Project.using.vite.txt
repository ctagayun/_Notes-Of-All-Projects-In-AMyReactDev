
1. Setting up using Vite
    - create a folder chitoreactdev
    - navigate to the folder where you want to create your React project
        cd C:\_AMyReactDev\chitoreactdev
    - create the React project where xxxx is the project name
        npm create vite@latest react-app-created-by-vite -- --template react

    - Now run:
        cd fakeapi
        npm install
        npm run dev (to see project running in dev)

    - type code . to go to Visual Studio Code

2. Adding bootstrapp
    npm install react-bootstrap bootstrap

3. Installing Axios
      - run npm install axios in VSCode terminal
